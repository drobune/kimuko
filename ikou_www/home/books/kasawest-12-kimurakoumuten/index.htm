関西の優良工務店
Part.1

 株式会社 木村工務店
株式会社木村工務店
得意分野のアピール、 技術の伝承にIT を有効活用
大阪市生野区小路東にある株式会社
木村工務店は、創業6 4 年という長い
歴史を持った工務店である。地下鉄千
日前線「小路」駅から5 分余りの場所
にある同社に3 代目で専務取締役の木
村貴一氏を訪ね、創業から現在に至る
経緯、業務の特色、これからの方向性
についてうかがった。
大阪市とともに歩んだ戦後の歴史

創業者は木村氏の祖父にあたる木村
精一氏で、1 9 3 7 年（昭和12 年） に
建築業木村商店としてこの地で創業さ
れた。昭和10 年代は、大阪市の東部
で木造長屋建築や文化住宅を多数建築
 		したという。戦後は1 9 4 9 年（昭和
2 4 年） にいち早く株式会社木村工務
店に組織を変更。大阪市の指名業者と
なり、昭和2 0 年代は戦災で焼失した
小学校の木造校舎の新築を始め、木造
作業所や復興住宅を施工。また、新制
中学校の開校と校舎の鉄筋コンクリー
ト化が始まり、19 5 1 年（昭和2 6 年）
戦後初の鉄筋校舎の1 つである生野中
学校を辛がけ、3 0 年代は大阪市の工
事を主力に鉄筋校舎を多数施工するな
ど、同社の歴史はまさに戦後の大阪市
の発展と共に推移してきたといえる。
また先見の明があつたので；あろう、
A L C の施工も他社に先がけて辛がけて
おり、関西で最初のA L C 建物の施工も
同社の仕事である。

設計・施工が6 0 ～ 7 0 ％、
設計事務所の仕事が3 0 － 4 0 ％

 原則的には社内にも設計担当者を抱
え、設計施工で展開しているが日建設
計のインテリアの仕事などはかなり以
前から手がけており、現在は、設計施
工が6 0 ～ 7 0 ％ 、設計事務所の仕事
が3 0 ～ 4 0 ％ だそうである。
住宅は新築が年間約1 5 棟以外にリ
フォームを多数辛がけている。集合住
宅は2 0 戸程度の規模のものを年間1
～2 棟、5 0 年代に一時止めていた公
共工事の受注も再開している。
古くから営業しているだけあつて、
木造なら刻みもプレカットもこなせる
大工をいまでも5 名抱えており、神社
仏閣の仕事も宇がけている。設計事務
所からも木造の腕を買われての仕事が
多いという。営業エリアは、車で1 時
間半程度の場所だそうである。
 		社員1 人1 台の
パソコン体制を実現
1 9 9 8 年（平成1 0 年）、品質管理の
システム化を区lることになり、第一段
階として2 0 0 0 年（平成1 2 年）春に
社員1 人1 台のパソコン体制とし、社
内LA N によるIT 化が実現された。サ
イボウズというグループウエアを活用
して品質管理システムの構築がスター
トしたのである。
同社としてどういう品質管理をする
のかということがいろいろ論じられた
結果、これまでの長い歴史の中で欠け

ていることを考え、社内での伝達の問
是喜を改めて考え直すことにしたという。
当初は、電子会議室を設け、クレーム
の報告から行うことにしたという。施
工材料、構造等の問題を3 ～ 4 人で実
験的に取組むことからスタートしたの
である。この時点ではまだ社員全員が
パソコンを使える状態ではなく、1 年
半後を目処に全員にパソコンを持たせ
るとなれば、パソコンの利便性をでき
るだけ前面に出して、パソコンを使お
うという雰匪l気づくりから始めたとい
う。
技術の伝承に活用

 熟練した技術者がいつまでもいてく
れるわけではない。看い世代に交代し、
技術者に頼れなくなっても今までの技
術を伝えていけるよう、技術を写真に
とって記録に残すことから始めようと
いうのである。アドバイスがもらえな
くなった時に助け合っていけるような
ツールに出来たらというのが木村専務
の考えである。クレームに対しても社
内でフォローしてリカバリーする体制
をつくることが大事なのでiぁる。
設計事務所の仕事の場合でも、品質
管理という面ではデジタルカメラで写
真を撮るなど、情報を共有することの
価値は大きい。

 		得意分野のアピールに
ホームページを活用
景気の良い時はいいが、不況が長引
く時代になると自分たちの得意とする
ポイントをアピールしていくことも重
要だと、19 9 8 年（平成1 0 年） に木
村専務が中心になつてホームページを
開設した。長い間この地で営業してき
た同社でも、いま改めて地場の再確認
も考えているという。ホームページに
小路スタイルというこの地の歴史や特
有の暮らし方、町の情報等を載せたペ
ージを作られたのもその一環である。
また、物作りの仲間ということで協力
会社も紹介、似顔絵や写真も掲載して
いるが顔の見える業者ということで、
建築主にとっての安心感も大きいとい
う。n .を単なる流行で終わらせるので
なく、業務上での有効な利用法を考え
ていこうと積極的な取り組みがなされ
ており、1 年後、2 年後の結果が楽し
みだともいえる
